# 抽卡模拟器 (Gacha Simulator)

一个功能完整的抽卡模拟器，模拟 Endfield 游戏的抽卡机制，支持多种抽卡策略和详细的统计分析。

## 功能特性

### 🎮 核心抽卡机制
- **软保底系统**：65抽开始概率递增，90抽必出6星
- **硬保底系统**：120抽必出当期UP角色
- **卡池规则**：支持每期UP角色配置
- **资源管理**：模拟石头资源的获取和消耗

### 📊 抽卡策略
- **抽到当期UP就收手**：抽到当期UP角色停止，歪了且有65抽资源继续
- **120抽硬保底**：确保120抽必出当期UP
- **120+30循环**：资源充足时目标120抽，不足时目标30抽

### 📈 统计分析
- **详细日志**：记录每池抽卡详情、资源使用和6星结果
- **策略对比**：多策略并行模拟，计算效率和收益
- **数据可视化**：平均获得6星数量、UP角色数量、资源利用率

### 🔧 技术特点
- **模块化设计**：清晰的分层架构
- **完整测试**：47个单元测试，100%通过
- **高性能**：支持大规模模拟（10000+次）
- **可扩展**：易于添加新的抽卡策略和规则

## 核心类说明

- **GachaEngine**：核心抽卡逻辑，处理概率计算和保底机制
- **CardPool**：卡池管理，包含UP角色和常驻角色
- **ResourceManager**：资源管理，处理石头的获取和消耗
- **GachaSimulator**：模拟器，执行抽卡流程和策略管理
- **StatisticsTracker**：统计追踪，记录抽卡结果和资源使用
- **各种Strategy**：不同抽卡策略的实现

## 资源配置

- 第1期：120石头
- 第2-3期：各80石头
- 第4期及以后：各60石头
- 总资源：1280石头

## 抽卡规则

1. **6星概率**：基础概率2%，65抽后开始递增
2. **UP概率**：6星中50%概率为当期UP
3. **硬保底**：120抽必出当期UP
4. **软保底**：90抽必出6星
